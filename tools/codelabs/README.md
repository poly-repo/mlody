# Codelabs

This directory contains **codelabs** for this repository.  
Each codelab is intended to be a focused, hands-on guide that walks
through a specific concept, tool, or workflow.

## Creating a New Codelab

If you’re here to **write** a codelab, start with this Codelabs tutorial:

👉 https://codelabs.solace.dev/codelabs/codelab-4-codelab/?index=..%2F..index#0

It covers structure, metadata, formatting, and the things you’ll
otherwise discover the hard way.
